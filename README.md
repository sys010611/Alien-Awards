## プロジェクト概要

成均館大学のゲーム開発サークルが主催したゲームジャムで制作したプロジェクトです。
ゲームのコンセプトは「宇宙人に正しい賞状を授与する」というもので、賞状の内容や宇宙人の見た目にはランダム性があり、ゲームが進むにつれて徐々に難易度が上昇するシステムを実装しました。
